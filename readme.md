# How to use

#### First,you need to install npm & electron.  for fedora,do:

    sudo dnf install nodejs
    npm install electron


#### Then clone this repository:

    git clone https://github.com/electron/electron

#### Last:

    electron ./ElecMD


Tested on fedora
